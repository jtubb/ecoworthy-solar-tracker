Solar dual axis tracker custom firmware design description

Based on an STC 15F2K60S2 (SOP28)

Programming header that connects to pins 15 and 16

Outputs:
1602A screen
Piezo buzzer
uln2003a darlington transistore that drives 4 relays, those relays trigger provide 12v two linera actuators W/E out, and N/S out.

Inputs
6 buttons, Set (B1), Quit(B2), North (B5), South (B6), East (B4), and West (B3)
IR sensor, unknown function, looks like for a remote control the vendor never developed.
wind speed sensor
Sun intensity sensor
E,W,S,N limit switches (no limit switches provided with kit? maybe soft sensing current draw?) - limit switches go to diodes that would be wired to pull the inputs to uln2003a down.
